# Software Quality Engineering Project

## Project Description
This project automates testing of the SauceDemo web application using Selenium WebDriver and Pytest.

## Technologies Used
- Python
- Selenium
- Pytest
- GitHub Actions

## Features Tested
- Login
- Inventory
- Cart
- Checkout
- Browser Launch

## Project Structure
selenium_automation/
pages/
tests/
conftest.py
requirements.txt

## Running Tests Locally

pip install -r selenium_automation/requirements.txt
pytest

## CI/CD

GitHub Actions automatically runs the Selenium test suite whenever code is pushed to the main branch or a pull request is created.
